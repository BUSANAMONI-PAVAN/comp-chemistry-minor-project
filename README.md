\# Anticancer Drug Solubility Prediction



\## Overview



This project focuses on analyzing and predicting the solubility of anticancer compounds using computational chemistry and machine learning techniques. Solubility is a crucial property in drug development because it directly affects the absorption, distribution, and effectiveness of a drug.



The project uses molecular data obtained from PubChem to study the solubility behavior of anticancer compounds and visualize patterns through data analysis.



---



\## Objectives



\* Analyze anticancer compound data.

\* Study the relationship between molecular properties and solubility.

\* Visualize solubility trends using plots.

\* Build a computational workflow for drug solubility analysis.



---



\## Dataset



The dataset used in this project is obtained from \*\*PubChem\*\* and contains molecular information related to anticancer compounds.



File used:



\* `PubChem\_compound\_anticancer.csv`
